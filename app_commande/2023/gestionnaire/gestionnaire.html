mode d'emploi

le generateur sert a cree un array javascript contenant une liste de toutes les elements et leurs prix il sert aussi a sauvegarder la liste pour une utilisation ulterieur et permet de gerer cete liste en supprimant rajoutant et modifier des elements.

menu

fichier : sert a charger les donné qui on éte soit sauvegardé dans le navigateur(navigateur) soit sauvegardé dans un fichier(parcourir). quand a l'option 'inserer liste' elle sert a rajouter une liste dans la liste courante.

enregistrer : enregistre les donné dans le navigateur ou telecharge le fichier contenant les donné.

generer page : genere la page html complete et lance son telechargement le fichier est a inserer dans le dossier principale.

tester : permet de tester le resultat.

vue reel : affiche les donné selon l'ordre d'affichage du fichier html.

deplacer categorie : permet de modifier la position d'une categorie par drag & drop.

deplacer article : permet de modifier la position d'un article dans sa categorie par drag & drop.

renomer categorie : permet de renomer une catégorie
